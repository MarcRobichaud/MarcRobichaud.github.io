Splash Royale

A multiplayer auto battler strategy game inspired by Clash Royal using the new multiplayer solution for unity: NetCode

Trailer

Team Structure & Roles

Solo Project. Networking, Animations, Movement, Unit Creation, Attacks, Dealing Damage, Area of Effect, Damage Overtime, On Death Logic, Map Creation, UI, Priority System, Player Inputs, Particle Effects, Mana System

Team Management

Agile Scrum methodology for an incremental and iterative project with Jira even if I was alone. Greatly facilitated the general organisation and planification of the project. A Game Design Document (GDD) was completed beforehand to have clear objectives and clear structure. Code Review every monday.

Challenges

Networking

Netcode for Game Objects, a new multiplayer solution for unity that was launch a few months before the start of the project. Documentation could not be found at first because it had recently moved and made the installation a bit tricky.

Netcode Usage: NetworkManager, Network Objects, Network Transform, Network Animations, Rpc Call, Network Variables

Units

The units had to be flexible, reusable code as much as possible and fast creation of new one with different abilities and behaviour

Solution: Modular Design using scriptable objects
